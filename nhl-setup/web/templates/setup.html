Terminal Setup Required

This server has detected that you are running the scoreboard for the first time. The initial setup requires interactive, system-level access and **must be run from a terminal on the Raspberry Pi**.

Please SSH into your Pi (as the pi user) and follow the prompts. To run tool manually,

/home/pi/nhl-led-scoreboard/sb-tools do_firstrun

After you follow the on-screen prompts and reboot, this page will automatically redirect to the Control Hub.
